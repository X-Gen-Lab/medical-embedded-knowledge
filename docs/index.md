# 医疗器械嵌入式软件知识体系

欢迎来到医疗器械嵌入式软件知识体系！

本知识库旨在为医疗器械嵌入式软件开发人员、质量工程师和监管人员提供全面、系统的学习资源。

## 主要内容

### 核心技术知识
- 嵌入式C/C++编程
- 实时操作系统（RTOS）
- 硬件接口
- 低功耗设计
- 信号处理与算法

### 医疗法规与标准
- IEC 62304 - 医疗器械软件生命周期过程
- ISO 13485 - 质量管理体系
- ISO 14971 - 风险管理
- FDA法规
- IEC 60601-1 - 医疗电气设备安全
- IEC 81001-5-1 - 网络安全

### 软件工程实践
- 需求工程
- 软件架构设计
- 编码规范（MISRA C、CERT C）
- 测试策略
- 配置管理
- 静态代码分析
