# 高级嵌入式系统

本模块涵盖医疗设备开发中的高级嵌入式系统主题，包括多核处理器编程、DMA技术、故障恢复机制和引导加载程序。

## 📚 学习路径

### 基础要求
- 熟悉C/C++编程
- 理解嵌入式系统基础
- 掌握RTOS概念
- 了解硬件架构

### 学习顺序

1. **[多核处理器编程](multicore-programming.md)** ⭐⭐⭐
   - 对称多处理（SMP）
   - 非对称多处理（AMP）
   - 核间通信机制
   - 负载均衡策略

2. **[DMA（直接内存访问）](dma.md)** ⭐⭐⭐
   - DMA控制器配置
   - 循环缓冲区设计
   - DMA与中断协作
   - 性能优化

3. **[看门狗与故障恢复](watchdog-recovery.md)** ⭐⭐⭐
   - 硬件看门狗机制
   - 软件看门狗实现
   - 故障检测策略
   - 安全状态设计

4. **[引导加载程序](bootloader.md)** ⭐⭐⭐
   - U-Boot配置
   - 安全启动流程
   - 固件更新机制
   - 故障恢复
