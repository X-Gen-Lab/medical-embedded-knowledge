# 入门指南

欢迎使用医疗器械嵌入式软件知识体系！本指南将帮助您快速开始学习。

## 针对不同角色的指南

- **[开发人员入门指南](for-developers.md)** - 为嵌入式软件开发人员设计，涵盖学习路径、技术资源和开发要点
- **[QA工程师入门指南](for-qa-engineers.md)** - 为质量保证工程师设计，涵盖测试策略、工具和法规要求
- **[监管事务专员入门指南](for-regulatory-affairs.md)** - 为监管事务专员设计，涵盖法规体系、认证流程和文档要求

## 如何使用本知识体系

1. 选择适合您角色的学习路径
2. 按照推荐顺序学习知识模块
3. 完成每个模块的自测问题
4. 参考实践案例加深理解
