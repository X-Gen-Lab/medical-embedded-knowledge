## 如何选择学习路径

根据你的职业角色和学习目标选择合适的学习路径：

- **嵌入式软件工程师**: 如果你负责医疗器械的软件开发和实现
- **质量保证工程师**: 如果你负责质量管理、测试和合规性验证
- **系统架构师**: 如果你负责系统设计、架构决策和技术选型
- **监管事务专员**: 如果你负责法规合规、认证申报和技术文档

## 学习建议

!!! tip "学习提示"
    - 每个学习路径都经过精心设计，建议按照推荐顺序学习
    - 必修模块是核心内容，选修模块可根据实际需求选择
    - 在关键节点设有检查点，帮助你评估学习效果
    - 可以根据实际情况调整学习节奏，但建议完成所有必修内容
